# 星际之门 · StarGate（备选）- 配置说明

## 配置项说明

### `bin_app` — 后台核心类型

选择代理服务使用的内核：

| 值 | 说明 |
|----|------|
| `v2ray-core` | V2Ray 内核，兼容性好，适合使用 VMess、Trojan 等常见协议 |
| `xray-core` | Xray 内核，**推荐使用**，支持更多新协议（如 VLESS+REALITY），性能略优 |

> 切换内核后需重启插件才能生效。

---

### `clear_mode` — 清除用户数据

| 值 | 说明 |
|----|------|
| `false` | 正常模式，**默认值**，保留用户配置和数据 |
| `true` | 每次插件启动时自动清除所有用户数据（账号、节点、规则等） |

> ⚠️ **警告：** 开启后每次重启都会清空所有配置，包括账号密码和导入的订阅。仅在需要重置时临时开启，完成后请立即改回 `false`。
>
> 注意：卸载插件本身不会删除用户数据，如需彻底清除请先开启此选项重启一次。

---

## 使用建议

备选版本的配置项相对精简，主要在 Web 控制台（端口 2017）内完成以下配置：

1. **透明代理实现方式**：推荐选择 `system tun` 模式用于长期稳定运行；`redirect` 模式兼容性更强但不适合长期使用。
2. **DNS 模式**：推荐选择 `Doh` 防止 DNS 污染；如网络较好可选 `Direct`。
3. **防火墙模式**：按照控制台页面提示选择，通常保持默认即可。

配置完成后点击「启动」即可。
