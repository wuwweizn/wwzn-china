# 🌐 WebTunnel - 内网穿透客户端

**WebTunnel** 是一个专为 Home Assistant 设计的高效内网穿透插件，支持 **amd64** 和 **aarch64** 架构。

## ✨ 核心功能

- ⚙️ **极简配置**：开箱即用，快速部署
- 🔄 **稳定穿透**：支持多种协议，稳定可靠
- 🔒 **安全加密**：采用先进加密技术，保障数据安全
- 🌍 **多场景应用**：满足远程访问、设备管理等需求
