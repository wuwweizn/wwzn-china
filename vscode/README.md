## 关于

这个插件运行 **Visual Studio Code**，允许你直接在浏览器中编辑 Home Assistant 配置，
并且可以直接嵌入到 Home Assistant 前端 UI。

Visual Studio Code 通过 `code-server` 作为远程服务运行，
提供完整的 VSCode 使用体验。

插件已预装并预配置了 **Home Assistant、MDI 图标和 YAML 扩展**，
这意味着自动补全功能开箱即用，无需额外配置。
