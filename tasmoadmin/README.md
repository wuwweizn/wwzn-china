集中管理你所有的 Sonoff-Tasmota 设备。

＃关于

TasmoAdmin（前称 SonWEB）是一个管理界面，可以集中管理你所有刷了 Tasmota 的 Sonoff 设备。
它的一些功能包括：

扫描你的网络并自动添加设备

快速、轻松地查看所有设备的状态

从一个界面配置所有设备

可同时对一个或多个设备进行 OTA（空中）固件升级

可以自动为你下载最新固件
